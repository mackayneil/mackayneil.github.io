Restaurant app prototype

Restaurant Mobile Application Prototype (look and feel only) project on Freelancer

The project

This project involved creating a prototype for a client on Freelancer.
The project details were as follows: "This application will be used by end users (i.e. restaurant guests) inside the restaurant to order their food instead of waiting for the waiter, it won't be used to order food delivery, it will only used inside the restaurant"

My goal

My goal for this project was to make an easy to use design that fulfilled all the clients requests.
These included having categories, different option for meal sizes and cooking preperations, an easy to find place order button, a countdown timer for the order and using a colourful design.

What I did

  • Created the design using Sketch
  • Imported the Sketch design into Figma to create the working prototype
  • Researched multiple colour palletes to find one that used a colourful design but still looked appealing
