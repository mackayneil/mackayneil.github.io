Challenge

The goal for this project was to redesign every page to give it a modernised look and feel.

Process

  • Conversed with company owner to better understand core business wants and needs
  • Created design mockups after looking closely at the key needs for each page
  • Participated in weekly meetings for approval or revision process
  • Coded approved designs using HTML, CSS and Javascript
  • Pushed changes to GitHub
  • Performed a QA process to test for any errors
  • Maintained page functionality and made changes where needed

Result

Updated a large database website consisting of over 35+ pages.
