Bonsai store template

This project is a template I designed for a Bonsai store (of course the images could be replaced to be used on any kind of store)

The project

This project involved researching some of the most popular designs and creating this design through Sketch.
After making the design I wrote all the code from scratch using a mobile first approach.

My goal

My goal for this project was to really emphasise the Bonsai species that are available at the store.

What I did

  • Designed the protoype using Sketch.
  • Wrote the code (HTML, CSS, Javascript) using Visual Studio Code.
  • Ensured that it worked accross multiple browsers and device sizes.
