Wild Leaf Product Page

Wild Leaf Kombucha is a Cape Town based Kombucha company that sells their product online as well as a variety of stores.

The project

This project involved redesigning the product page on Shopify.
I used Sketch to design the product page, Visual Studio Code to create it and then I replaced the code on the Shopify product page with my own.

My goal

My goal for this page was to make the layout look more modern as well as be responsive on different mobile screens.
The original product page on Shopify had no structure and was a very top to bottom layout which wasn't responsive on mobile.

What I did

  • Designed the product page using Sketch.
  • Wrote the code on Visual Studio Code to create the page.
  • Replaced all the code on the Shopify product page with my own, ensuring that the products were still easily editable on the front-end of the Shopify platform.
  • Created and edited all the product photos.
