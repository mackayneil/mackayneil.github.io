Tiffany Sosa

This project is Tiffany's portfolio for her graphic design and artwork that she uses to present to potential clients.

The project

This project involved rebuilding a portfolio website based off an existing one. The existing one had issues with missing links, reponsiveness on screen sizes and overall layout issues.
I decided it would be easier to rebuild the entire website which helped give it a fresh look.

My goal

My goal for this project was to highlight Tiffany's artwork and graphic design work. To make it easy for users to see what she is capable of creating and to be able to contact her easily if they wish.

What I did

  • Rebuilt the front-end of the website after making improvements to the layout, repsonsiveness and navigation of the already existing portfolio website.
  • Created the wireframe and a prototype using Adobe XD.
  • Integrated the website onto WordPress.
