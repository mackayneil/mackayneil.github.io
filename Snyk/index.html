Challenge

Each of these pages shown had their own challenges to navigate, but as a summary my goal was to ensure that users could onboard their cloud environments, view their issues, and ignore the issues if they wanted to.

Process

  • Closely reviewed existing patterns within Snyk and worked with other designers to implement new patterns for the needs of my work and any use cases there may be else where within the platform.
  • Worked closely with stakeholders and managers to get design feedback early on.
  • Iterated on the designs after getting feedback from users as well as internal people.
  • Collaborated with the frontend team to build components needed.
  • Implemented the designs myself using Vue3 and wrote tests for new components and pages.
  • Updated any features that needed to be updated after getting user feedback.
  • Added additional features that weren't part of the MVP when time allowed.

Result

Snyk did not have a Cloud product before the acquisition of Fugue. This work helped bring over some key features as well as new ones from Fugue into Snyk.
