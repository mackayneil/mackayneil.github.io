Earth Fair Market

Earth Fair Market is a local market place in Cape Town where vendors can sell their goods to customers.

The project

This project involved creating a prototype of a revamped version of their website.
I used Figma to create a mobile and a desktop version of the website, which the client was then able to view.

My goal

My goal for this project was to redesign their current website to make it more user friendly and responsive.
I focused on creating a simple design that was easy to navigate and still displayed all the relevant information to users.

What I did

  • Built a working prototype using Figma for both mobile and desktop.
  • Redesigned the entire layout of the website.
  • Used a color palette that matched the current colors of the business.
