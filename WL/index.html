Wild Leaf Kombucha

A Cape Town Kombucha Brewing company.

  • Home page
  • Info page
  • Product page
Challenge

To create an ecommerce store that is user friendly and easy to make purchases.

Process

  • Reseached ecommerce websites and competitors to better understand the brand and industry
  • Developed a couple of low fidelity wireframes focusing on the layout of each page
  • Created a new colour palette and made a high fidelity wireframe
  • Built the front-end of all the pages using HTML and CSS on Shopify
  • Tested pages for responsiveness across multiple device sizes
  • Set up the backend of the Shopify store which included uploading all products, shipping details, activating payment methods
  • Created and edited all the product photos using Photoshop

Result

The end product is a functional website that provides brand and product information to customers and allows them to purchase online.
